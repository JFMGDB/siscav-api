# Instruções para Adicionar Evidência de Processo de Detecção

## Imagem Necessária

Para completar a apresentação, é necessário adicionar uma imagem que mostre o processo de detecção acontecendo em tempo real.

## Nome do Arquivo

A imagem deve ser salva com o seguinte nome:
```
evidencia-processo-deteccao-tempo-real.png
```

## Localização

A imagem deve ser colocada na pasta:
```
docs/presentation/assets/images/
```

## Conteúdo Esperado da Imagem

A imagem deve mostrar:
- Frame de vídeo capturado pela câmera
- Placa veicular detectada e destacada com retângulo
- Texto extraído exibido acima da placa (ex: "PLACA: ABC1234")
- Status de autorização (ex: "STATUS: Authorized" ou "STATUS: Denied")
- Interface visual do sistema IoT funcionando em tempo real

## Como Capturar a Imagem

1. Execute o sistema IoT (`apps/iot-device/main.py`)
2. Posicione uma placa na frente da câmera
3. Capture a tela quando o sistema detectar a placa
4. Salve a imagem como `evidencia-processo-deteccao-tempo-real.png`
5. Coloque na pasta `docs/presentation/assets/images/`

## Formato Recomendado

- Formato: PNG (para melhor qualidade)
- Resolução: Mínimo 1280x720 pixels
- Tamanho: Otimizado para web (máximo 2MB)

## Slide na Apresentação

A imagem será exibida no **Slide 4** da apresentação, logo após a "Proposta de Solução", mostrando o sistema em funcionamento.

---

**Nota**: Se a imagem já existir com outro nome, renomeie para `evidencia-processo-deteccao-tempo-real.png` e coloque na pasta indicada.
