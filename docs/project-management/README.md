# Project Management

Esta seção contém documentação relacionada ao gerenciamento de projeto, status de tarefas e rastreamento de progresso.

**Nota (2026):** cards e status podem referir `apps/iot-device/` ou `arduino/` — esses caminhos foram removidos do tree atual. Trate como histórico; o escopo ativo do repositório é a API em `apps/api/src/`.

## Documentos Disponíveis

### [Status das Tarefas](./backend-tasks-trello-status.md)
Mapeamento completo do status de implementação dos requisitos funcionais (RF) e não funcionais (RNF) do sistema, incluindo:
- Tarefas concluídas
- Tarefas em andamento
- Tarefas pendentes
- Tarefas bloqueadas/aguardando
- Evidências de implementação

### [Formato de Cards do Trello](./trello-cards-format.md)
Formato padronizado para criação de cards no Trello, facilitando o rastreamento e organização das tarefas do projeto.

## Uso

Estes documentos são úteis para:
- Acompanhamento do progresso do projeto
- Identificação de dependências entre tarefas
- Planejamento de sprints
- Documentação de decisões de projeto

## Manutenção

Estes documentos devem ser atualizados regularmente conforme o progresso do desenvolvimento:
- Atualizar status de tarefas conforme implementação
- Adicionar novas tarefas identificadas
- Documentar bloqueios e dependências
