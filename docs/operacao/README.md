## Requisitos Mínimos

- Python 3.10, 3.11 ou 3.12 (evitar 3.13 e 3.14)
- Windows 10/11, Linux (Ubuntu 20.04+), ou macOS
- 4GB RAM mínimo, 8GB recomendado
- Câmera USB ou IP
