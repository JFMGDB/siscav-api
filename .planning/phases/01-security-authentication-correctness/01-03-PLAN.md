## objective

Add **`is_admin: bool`** to **`users`** (**default `False`**) via **Alembic**, introduce **`get_current_admin_user`** (JWT via existing **`get_current_user`** path + **`user.is_admin is True`**; non-admin → **403**), apply it to routes documented as administrator-only (**`GET /api/v1/access_logs/images/{image_filename}`**, **`POST /api/v1/gate/trigger`**), and align **`FastAPI` `description`** and route docstrings with runtime checks (**D-09**). Document first-admin bootstrap (**D-10**).

**Purpose:** Make privileged surface claims in OpenAPI/docstrings honest (**SEC-03**). Runs in **wave 2** so device ingest and production secret hygiene land first without conflicting file edits.

**Output:** Migration + model + deps + wired endpoints + tests + short bootstrap note under **`docs/api/`**.

## must_haves

1. DB column **`users.is_admin`** exists (Ale **`Boolean`**, **`server_default=false`** or equivalent), ORM **`User.is_admin`** mapped, downgrade drops column.
2. **`get_current_admin_user`** returns **`User`** only when **`is_admin`** is true; otherwise **403** with generic credential/forbidden wording consistent with **`deps.py`**.
3. **`get_access_log_image`** and **`trigger_gate`** use **`Depends(get_current_admin_user)`** instead of **`get_current_user`**.
4. OpenAPI top-level **`description`** in **`main.py`** states that gate trigger and access-log image download require **admin** JWT (Portuguese OK); route docstrings do not claim admin-only unless enforced.
5. **`docs/api/README.md`** (or same doc tree executor chooses) contains SQL snippet to set **`is_admin = true`** for first operator (**D-10**); **`seed_demo.py`** comment references it if seed does not auto-promote.
