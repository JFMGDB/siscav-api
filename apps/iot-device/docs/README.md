# Documentação do Dispositivo IoT

Esta pasta contém toda a documentação relacionada ao dispositivo IoT do SISCAV, incluindo instalação, configuração, uso e troubleshooting.

## Estrutura

### Documentos Principais

- **[README.md](./README.md)** - Resumo executivo do dispositivo IoT
- **[installation.md](./installation.md)** - Guia de instalação e configuração
- **[troubleshooting.md](./troubleshooting.md)** - Soluções para problemas comuns
- **[refactoring.md](./refactoring.md)** - Documentação da refatoração do dispositivo
- **[demo-guide.md](./demo-guide.md)** - Guia completo de demonstração
- **[demo-evaluation-guide.md](./demo-evaluation-guide.md)** - Guia de avaliação de desempenho

### Hardware

A pasta `hardware/` contém documentação sobre o Arduino e componentes físicos:

- **[arduino.md](./hardware/arduino.md)** - Documentação do firmware Arduino
- **[project-definition.md](./hardware/project-definition.md)** - Definição do projeto Arduino
- **[assembly-guide.md](./hardware/assembly-guide.md)** - Guia de montagem e demonstração

### Archive

A pasta `archive/` contém documentos históricos e obsoletos:

- **python312-installation.md** - Guia de instalação do Python 3.12 (obsoleto, já resolvido)
- **python312-regression.md** - Documentação da regressão para Python 3.12
- **python314-final-solution.md** - Solução final para Python 3.14
- **numpy-error-solution.md** - Solução rápida para erro do NumPy

## Início Rápido

1. Leia o [guia de instalação](./installation.md)
2. Configure o ambiente seguindo as instruções
3. Consulte o [guia de demonstração](./demo-guide.md) para executar o sistema
